# GravityShiftRep

"GravityShift" – это 3D-платформер-головоломка, где игрок должен проходить минималистичные уровни-комнаты.
Ключевая механика – мгновенное переключение вектора гравитации (вверх/вниз), но только при контакте с поверхностью.
Горизонтальное движение (A/D) ограничено, а прыжка (пробел) нет – вместо него происходит переключение гравитации.

Платформа: PC (Windows 10+).
Минимальные требования: CPU Intel Core i3 / AMD Ryzen 3, 4GB RAM, GPU Intel HD Graphics 4000+ / эквивалент, 100MB HDD.

Для запуска проекта не требуется установка Unity (для готовой сборки).
Вариант А (запуск готовой игры):
1. Перейдите в папку /Builds в архиве проекта.
2. Запустите исполняемый файл GravityShift.exe.
Вариант Б (запуск исходного кода):
1. Установите Unity Hub и версию Unity 2022.3 LTS.
2. В Unity Hub нажать «Add» и выбрать корневую папку проекта.
3. Открыть проект.
4. В окне Project перейти в Assets/Scenes и открыть сцену MainMenu.
5. Нажать кнопку Play в редакторе.
