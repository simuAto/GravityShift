# GravityShiftRep

"GravityShift" – это 3D-платформер-головоломка, где игрок должен проходить минималистичные уровни-комнаты.
Ключевая механика – мгновенное переключение вектора гравитации (вверх/вниз), но только при контакте с поверхностью.
Горизонтальное движение (A/D) ограничено, а прыжка (пробел) нет – вместо него происходит переключение гравитации.

Платформа: PC (Windows 10+).

Минимальные требования: CPU Intel Core i3 / AMD Ryzen 3, 4GB RAM, GPU Intel HD Graphics 4000+ / эквивалент, 100MB HDD.

Для запуска проекта не требуется установка Unity:
1. Перейдите в релизы.
2. Скачайте и извлеките папку.
3. Запустите исполняемый файл GravityShift.exe.
